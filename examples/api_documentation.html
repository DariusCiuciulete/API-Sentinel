REST API Documentation

User Management API

Base URL: https://api.example.com/v1

Get All Users

Endpoint: GET /users

Description: Retrieve all users from the system

Authentication: Bearer Token

GET https://api.example.com/v1/users

Create User

Endpoint: POST /users

Description: Create a new user account

POST https://api.example.com/v1/users

Get User by ID

Endpoint: GET /users/{id}

Description: Retrieve a specific user by ID

GET https://api.example.com/v1/users/123

Product API

Base URL: https://api.example.com/v2

List Products

Endpoint: GET /products

Description: Get all available products

GET https://api.example.com/v2/products

Create Product

Endpoint: POST /products

Description: Add a new product to inventory

POST https://api.example.com/v2/products

Update Product

Endpoint: PUT /products/{id}

Description: Update existing product details

PUT https://api.example.com/v2/products/456

Delete Product

Endpoint: DELETE /products/{id}

Description: Remove a product from inventory

DELETE https://api.example.com/v2/products/456
